id选择器、类选择器 
		

#labelid{
	text-align: justify;/*justify*/
	font-family: "Gill Sans", "Gill Sans MT", "Myriad Pro", "DejaVu Sans Condensed", Helvetica, Arial, "sans-serif";/*字体：gill sans*/
	color: yellow;/*颜色：黄色*/
}
.labelclass{
	text-align: center;/*居中*/
	font-family: "楷体";/*字体：楷体*/
	color: aqua;/*颜色：亮蓝色*/
}

结论：ID选择器 (优先级) > 类选择器。

类选择器、标签选择器 
		

.labelclass2{
	font-family: "华文彩云";
	font-size: 30px;
	color:violet;
}
label{
	font-family: "黑体";
	font-size: 25px;
	color: aqua;
}

结论：类选择器 (优先级) > 标签选择器。

子选择器、后代选择器 
		

div > label{
	font-family: "隶书";
	font-size: 40px;
	color: aliceblue;
}
div label{
	font-family: "楷体";
	color:aqua;
	font-size: 30px;
}

结论：子选择器 (优先级) > 后代选择器。

后代选择器、伪类选择器 
		

div label{
	font-family: "楷体";
	color:aqua;
	font-size: 30px;
}
label:first-child{
	font-size: 45px;
	color: red;
	font-family: "方正舒体";
}

结论：伪选择器 (优先级) > 标签选择器。
