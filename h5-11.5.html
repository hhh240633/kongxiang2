1.table布局

    当年主流的布局方式，第一种是通过table tr td布局

    示例： 

table{ 
	width: 800px; 
	height: 300px; 
	border-collapse: collapse; 
} 
.left{ 
	background-color: red; 
} 
.right{ 
	background-color: blue; 
}

页面效果： 文字自动垂直居中，很方便 同样可以设置左右的width

2.类比表格的table class

    示例： 

.table1{
	display: table;
	width: 800px;
	height: 300px;
	position:relative;
	left: 50%;
 	margin-left: -400px;
}
.tb_row{
	display: table-row;
}
.tb_cell{
	display: table-cell;
	vertical-align: middle;
}
.left1{
	background-color: red;
}
.right1{
	background-color: blue;

3.flexbox布局

    两列布局

    关键：父级元素设置display：flex 

    示例： 

.parent{
	width: 800px;
	height: 300px;
	display: flex;
	position:relative;
    left: 50%;
    margin-left: -400px;
}
.left3{
	width: 400px;
	height: 100%;
	background: red;
}
.right3{
	flex: 1;
	height: 100%;
	background: blue;
}
左
右

4.float布局

    两列布局——左侧定宽，右侧自适应

    关键：左侧设置float：left

    右侧：margin-left： leftWidth 

    示例： 

.parent2 {
  width: 800px;
  height: 200px;
  position: relative;
  left: 50%;
  margin-left: -400px;
}
.left4 {
  width: 200px;
  height: 100%;
  float: left;
  background-color: red;
}
.right4 {
  height: 100%;
  margin-left: 200px;
  background-color: blue;
}
