CSS盒模型

  • 简单模型实现代码如下：
    • .div-1 {
    background-color:blueviolet;
    width: 300px;
    border: 15px solid aqua;
    border-radius: 10px;
    padding: 50px;
    margin: 20px;
    text-align: center;
    overflow-x: hidden;
    overflow-y: hidden;
	box-shadow: 10px 10px 5px #888888;
}

    CSS 盒模型（框模型）实质上是一个包装每个 HTML 元素的盒子。它包括：边框、内边距（填充）、外边距以及实际的内容。
