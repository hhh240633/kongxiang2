Web实验3重点知识汇总

1、HTML5 - 新特性

    HTML5 的一些最有趣的新特性：

  • 新的语义元素，比如 header, footer, article,section；

  • 新的表单控件，比如数字、日期、时间、日历和滑块；

  • 强大的图像支持（借由 canvas 和 svg）；

  • 强大的多媒体支持（借由 video 和 audio）；

  • 强大的新 API，比如用本地存储取代 cookie。

2、HTML5 - 新增的语义化标记元素

    元素及其说明：

  • header：页面或页面中某一个区块的页眉，一般为导航信息；

  • footer：页面或页面中某一个区块的页脚；

  • section：页面中的一块区域，通常由内容和标题组成；

  • article：独立的内容块，可独立于页面其他内容适用；

  • aside：非正文内容，独立于页面的主要内容；

  • nav：作为页面导航的辅助内容。

3、HTML5 - 新增的功能元素

    元素及其说明：

  • hgroup：用于对整个页面或页面中一个内容区块的标题进行组合；

  • video：定义视频，例如电影片段或其他视频流；

  • audio：定义音频，例如音乐或其他音频流；

  • output：表示不同类型的输出，例如脚本的输出；

  • keygen：表示生成密钥；

  • 等等...（详见课本第6页）

4、HTML5 - 被删元素

    以下 HTML 4.01 元素已从 HTML5 中删除：

  • acronym；

  • applet；

  • basefont；

  • big；

  • center；

  • dir

  • font

  • frame

  • frameset

  • noframes

  • strike

  • tt

5、HTML5 - 浏览器支持

  • 所有现代浏览器都支持 HTML5。

  • 此外，所有浏览器，不论新旧，都会自动把未识别元素当做行内元素来处理。

  • 正因如此，您可以帮助老式浏览器处理”未知的“ HTML 元素。

  • 注释：您甚至可以教授石器时代的 IE6 如何处理未知的 HTML 元素。
