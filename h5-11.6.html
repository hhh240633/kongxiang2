CSS3三列布局



float布局

我是第一列！

代码如下:

   .float-left {
            width: 400px;
            height: 300px;
            background-color: red;
            float: left;
        }

我是第三列！

代码如下:

   .float-right {
            height: 300px;
            background-color: yellow;
            margin-left: 470px;
            margin-right: 470px;
        }

我是第二列！

代码如下:

   .float-center{
            width: 400px;
            height: 300px;
            background-color:#3F3;
            float: right;
        }
